### 입력 함수 정리

    scanf() 함수는 \n(줄바꿈문자)를 가져오지 않고, 마지막에 \0(널문자)를 붙인다.
    gets() 함수는 \n(줄바꿈문자)까지 가져오고, \n을 \0으로 대체 한다.
    fgets() 함수는 \n(줄바꿈문자)까지 가져오고, 추가적으로 \0을 붙인다.

    
    scanf : OOOOO(엔터) ------------> OOOOO\0
    gets  : OOOOO(엔터) ---------> OOOOO(엔터) ------------> OOOOO\0
    fgets : OOOOO(엔터) ------------> OOOOO(엔터)\0
